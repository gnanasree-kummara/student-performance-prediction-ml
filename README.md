# Student Performance Prediction using Machine Learning

## Overview
This project predicts student academic performance using machine learning techniques based on academic and attendance data.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Random Forest Classifier

## Features
- Data preprocessing and feature engineering
- Model training and evaluation
- Achieved approximately 90% prediction accuracy

## Dataset
Academic and attendance-related student data (preprocessed).

## Outcome
Helps identify students who may require additional academic support.
